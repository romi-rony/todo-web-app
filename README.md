# Todo Web App

A Todo web app with add, delete, complete tasks and summary counts using JavaScript.

## Features
- Add new todo tasks
- Delete individual tasks
- Mark tasks as completed
- Summary counts for:
  - Total tasks
  - Completed tasks
  - Uncompleted tasks
  - Deleted tasks

## Project Structure
The project includes multiple JavaScript implementations to demonstrate different approaches:

- `basic.js` – Basic DOM manipulation without using arrays
- `array.js` – Todo management using arrays
- `completeTodo.js` – Full implementation with state management and summary counts

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## Purpose
This project was built to understand:
- DOM manipulation
- Event handling
- State management using arrays
- Incremental feature development
